# Answers.com # 
Like the name suggests, Answers.com is a social website where people can come to get the answers to lifeâ€™s questions. Seeking explanation is an inherent side effect of being human and we seek to aid that pursuit of knowledge by connecting your questions to people around the world who may be knowledgeable on the topics you're interested in.

Answers.com allows users to:
* Ask a question
* Answer a question
* Up or downvote answers to those questions

## Full Stack ## 
* Node.js and Express.js used to create a RESTful API.

* Handlebars.js used as the templating engine. 

* MySQL and Sequelize ORM used for the database.

* GET and POST routes used for retrieving and adding new data.

* Deployed using Heroku.
